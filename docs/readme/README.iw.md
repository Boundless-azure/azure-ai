## 🚀 חזון ומטרות

**"XiaoLan"** הוא פורטל אינטראקציה מבוסס בינה מלאכותית מהדור הבא, שנועד להחליף פעולות אינטרנט מסורתיות מבוססות קליקים באינטראקציות בשפה טבעית. הוא מאפשר למשתמשים לנהל נתונים, לשלוט בממשקים ולבצע לוגיקה עסקית ישירות באמצעות שיחה.

תכונות ליבה:
*   **צמיחה עצמית**: הבינה המלאכותית מייצרת באופן אוטונומי תוספי קוד המשתלבים בחזרה בפלטפורמה, ויוצרים לולאת שיפור מתמדת.
*   **יצירה מבוקרת**: משתמש ב-"אילוצי יצירה + אוטובוס עסקאות (HookBus)" כדי להבטיח שהקוד שנוצר על ידי הבינה המלאכותית הוא בטוח, בעל תלות נמוכה וניתן לביקורת.
*   **סטנדרטים לביצוע AI**: פרוטוקולי שליטה סטנדרטיים לרכיבי צד-לקוח המאפשרים לבינה המלאכותית לבצע מניפולציות מדויקות על ממשק המשתמש.

## 🛠️ ארכיטקטורה טכנית

### 1. צד לקוח (Frontend Web Client)
*   **Tech Stack**: Vue 3 (Composition API) + TypeScript + Tailwind CSS + Pinia.
*   **עיצוב מודולרי**:
    *   מודול `Agent`: ליבת אינטראקציית הצ'אט.
    *   מודול `IM`: ממשק משתמש ותקשורת מופרדים באמצעות `useImChatAdapter`.
*   **חווית אינטראקציה**:
    *   **קלט רב-מודאלי**: הקלטה קולית (צורת גל בזמן אמת/ממשק מינימליסטי), טקסט, אימוג'י, תצוגה מקדימה של תמונות, טיפול בקבצים.
    *   **אנשי קשר חכמים**: קיבוץ אוטומטי (סוכני AI/קבוצות/אנשי קשר) עם מיון לפי Pinyin/ראשי תיבות.
    *   **תגובה בזרם (Streaming)**: תמיכה מלאה ב-SSE עם רינדור Markdown בזמן אמת.

### 2. צד שרת (Backend)
*   **תזמור שירותים**: שירותים מודולריים (מבוסס NestJS).
*   **קריאת פונקציה (Function Call)**: אינטגרציית פונקציות LLM טבעית עם הזרקת כלים דינמית.
    *   `PluginOrchestratorService`: תזמור תוספים.
    *   `MysqlReadonlyService`: שאילתות מסד נתונים מאובטחות לקריאה בלבד.
*   **HookBus**: האוטובוס המרכזי המחבר בין פעולות צד-לקוח ללוגיקת צד-שרת, ומבטיח עקביות וסדר אירועים.

## 📅 מפת דרכים (Roadmap)

### ✅ הושלם (שלב 1: יסודות ואינטראקציה)
- [x] **מסגרת צד-לקוח**: הגדרת Vue 3 + Tailwind מודולרית.
- [x] **אינטראקציית צ'אט**: רינדור הודעות בזרם, תמיכה ב-Markdown.
- [x] **קלט קולי**: ממשק הקלטה מינימליסטי עם צורת גל עוצמת שמע בזמן אמת.
- [x] **תמיכה רב-מודאלית**: בוחר אימוג'י, תצוגה מקדימה של תמונות בנייד/מחשב, אינטגרציה עם מערכת הקבצים.
- [x] **ספר כתובות חכם**: אסטרטגיית קיבוץ (סוכני AI > קבוצות > אנשי קשר) ומיון Pinyin.
- [x] **בינאום (Internationalization)**: תמיכה ב-i18n.

### 🚧 בתהליך (שלב 2: יכולות ליבה של AI)
- [ ] **פעולות כתיבה ל-DB**: ביצוע SQL מאובטח עם רשימה לבנה וביקורת.
- [ ] **הרשאות פרטניות**: בקרת גישה ברמת שורה/עמודה.
- [ ] **אינטגרציית HookBus**: אוטובוס אחיד לפעולות צד-לקוח וצד-שרת.
- [ ] **לולאת צמיחה עצמית**: זרימת עבודה של "תכנון -> יצירה -> בדיקה -> פריסה".

### 🔮 עתיד (שלב 3: אינטגרציה עמוקה)
- [ ] **Auto-CRUD**: יצירת דפי ניהול מתוך סכמת ה-DB.
- [ ] **שיתוף פעולה בין סוכנים**: תזמור משימות מורכב.
- [ ] **תוספים עסקיים**: ניתוח לקוחות, דוחות אוטומטיים וכו'.
