# Azure AI (XiaoLan)

ภาษา: [English](/docs/readme/README.en.md) · [中文](/docs/readme/README.zh-CN.md)

## 🚀 วิสัยทัศน์และเป้าหมาย

**"XiaoLan"** เป็นพอร์ทัลการโต้ตอบ AI รุ่นใหม่ที่ออกแบบมาเพื่อแทนที่การดำเนินการบนเว็บแบบคลิกแบบดั้งเดิมด้วยการโต้ตอบด้วยภาษาธรรมชาติ ช่วยให้ผู้ใช้สามารถจัดการข้อมูล ควบคุมอินเทอร์เฟซ และดำเนินการตรรกะทางธุรกิจผ่านการสนทนาโดยตรง

คุณสมบัติหลัก:
*   **การเติบโตด้วยตนเอง (Self-Growth)**: AI จะสร้างปลั๊กอินโค้ดโดยอัตโนมัติซึ่งรวมกลับเข้าไปในแพลตฟอร์ม ทำให้เกิดวงจรการปรับปรุงอย่างต่อเนื่อง
*   **การสร้างแบบควบคุม (Controlled Generation)**: ใช้ "ข้อจำกัดการสร้าง + บัสธุรกรรม (HookBus)" เพื่อให้แน่ใจว่าโค้ดที่สร้างโดย AI นั้นปลอดภัย มีการพึ่งพาต่ำ และสามารถตรวจสอบได้
*   **มาตรฐานการดำเนินการ AI**: โปรโตคอลการควบคุมส่วนประกอบส่วนหน้าที่เป็นมาตรฐาน ช่วยให้ AI สามารถจัดการ UI ได้อย่างแม่นยำ

## 🛠️ สถาปัตยกรรมทางเทคนิค

### 1. ส่วนหน้า (Frontend Web Client)
*   **Tech Stack**: Vue 3 (Composition API) + TypeScript + Tailwind CSS + Pinia
*   **การออกแบบโมดูลาร์**:
    *   โมดูล `Agent`: แกนหลักของการโต้ตอบการแชท
    *   โมดูล `IM`: แยก UI และการสื่อสารผ่าน `useImChatAdapter`
*   **ประสบการณ์การโต้ตอบ**:
    *   **อินพุตหลายรูปแบบ**: การบันทึกเสียง (รูปคลื่นเรียลไทม์/UI แบบมินิมอล), ข้อความ, อีโมจิ, ดูตัวอย่างรูปภาพ, การจัดการไฟล์
    *   **สมุดรายชื่ออัจฉริยะ**: การจัดกลุ่มอัตโนมัติ (AI Agents/กลุ่ม/ผู้ติดต่อ) พร้อมการเรียงลำดับ Pinyin/อักษรย่อ
    *   **การตอบสนองแบบสตรีมมิ่ง**: รองรับ SSE เต็มรูปแบบพร้อมการเรนเดอร์ Markdown แบบเรียลไทม์

### 2. ส่วนหลัง (Backend Server)
*   **การจัดการบริการ**: บริการแบบโมดูลาร์ (ใช้ NestJS)
*   **Function Call**: การรวมฟังก์ชัน LLM แบบเนทีฟพร้อมการฉีดเครื่องมือแบบไดนามิก
    *   `PluginOrchestratorService`: การจัดการปลั๊กอิน
    *   `MysqlReadonlyService`: การสืบค้นฐานข้อมูลแบบอ่านอย่างเดียวที่ปลอดภัย
*   **HookBus**: บัสกลางที่เชื่อมต่อการกระทำส่วนหน้าและตรรกะส่วนหลัง เพื่อให้มั่นใจถึงความสม่ำเสมอและลำดับของเหตุการณ์

## 📅 แผนงาน (Roadmap)

### ✅ เสร็จสมบูรณ์ (ระยะที่ 1: รากฐานและการโต้ตอบ)
- [x] **Frontend Framework**: การตั้งค่า Vue 3 + Tailwind แบบโมดูลาร์
- [x] **การโต้ตอบการแชท**: การเรนเดอร์ข้อความแบบสตรีมมิ่ง, รองรับ Markdown
- [x] **อินพุตเสียง**: UI การบันทึกแบบมินิมอลพร้อมรูปคลื่นระดับเสียงแบบเรียลไทม์
- [x] **การสนับสนุนหลายรูปแบบ**: ตัวเลือกอีโมจิ, ดูตัวอย่างรูปภาพบนมือถือ/พีซี, การรวมระบบไฟล์
- [x] **สมุดที่อยู่อัจฉริยะ**: กลยุทธ์การจัดกลุ่ม (AI Agents > กลุ่ม > ผู้ติดต่อ) และการเรียงลำดับ Pinyin
- [x] **ความเป็นสากล**: รองรับ i18n

### 🚧 กำลังดำเนินการ (ระยะที่ 2: ความสามารถหลักของ AI)
- [ ] **การดำเนินการเขียน DB**: การดำเนินการ SQL ที่ปลอดภัยพร้อม Whitelisting และการตรวจสอบ
- [ ] **สิทธิ์แบบละเอียด**: การควบคุมการเข้าถึงระดับแถว/คอลัมน์
- [ ] **การรวม HookBus**: บัสรวมสำหรับการกระทำส่วนหน้า-ส่วนหลัง
- [ ] **วงจรการเติบโตด้วยตนเอง**: เวิร์กโฟลว์ "วางแผน -> สร้าง -> ทดสอบ -> ปรับใช้"

### 🔮 อนาคต (ระยะที่ 3: การรวมเชิงลึก)
- [ ] **Auto-CRUD**: สร้างหน้าการจัดการจากสกีมา DB
- [ ] **การทำงานร่วมกันของหลาย Agent**: การจัดการงานที่ซับซ้อน
- [ ] **ปลั๊กอินธุรกิจ**: การวิเคราะห์ลูกค้า, การรายงานอัตโนมัติ ฯลฯ

## 🚦 เริ่มต้นอย่างรวดเร็ว

1. **ติดตั้ง Dependencies**
```bash
npm install
```

2. **เริ่มเซิร์ฟเวอร์การพัฒนา**
```bash
npm run dev
```

3. **สร้างสำหรับใช้งานจริง (Production)**
```bash
npm run build
```
